2. 原点は、絶対的に存在するものではない

ChatGPT

次は 座標変換 でよい。そして、その前提としての理解はかなり合っている。

点とベクトルは、数学的対象として扱う。基底・座標・行列は、それらを測る・記述する・変換するための道具である。

ただし、行列は単なる道具というより、「変換そのもの」を表す数学的対象でもある。

物理的には、速度が一定なら、

$$\Delta x = v\Delta t$$

なので、ある時間だけ切り取った運動は、変位ベクトルとして表せる。

ただし相対性理論では、単なる空間内のベクトルではなく、いつ・どこで起きたか、つまり事象 event を扱う必要がある。

3. 相対性理論では、時間を含む座標で扱う

相対性理論では、

$$(x,t)$$

または、

$$(ct,x)$$

のように時間を含む座標で扱う。

相対性理論では、空間だけを x → 2x のように変形するのではなく、空間座標と時間座標を混ぜる。

$$x' = \gamma(x - vt)$$
$$t' = \gamma\left(t - \frac{vx}{c^2}\right)$$

列車の光の話は、空間が前後に単純に伸び縮みする話ではなく、観測者によって「同時刻の切り方」が変わる話である。

原点は、絶対的に存在するものではなく、座標系を設定するために選ぶ基準である。相対性理論では、原点は空間上の点というより、ある観測者が「ここで、いま」と呼ぶ事象になる。

原点とは、世界の中心ではなく、座標系を始めるために選ばれた基準である。
